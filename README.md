FreelanceHub DBMS Project

A database-driven freelance marketplace system.

Features
- User registration
- Project posting
- Bidding system
- Contract management
- Milestone tracking
- Escrow payment system

 Tech Stack
- MySQL
- Node.js
- Express
- HTML/CSS/JS

 How to Run

1. Import schema.sql in MySQL
2. Run backend:
   cd backend
   npm install
   node server.js

3. Open frontend/index.html
